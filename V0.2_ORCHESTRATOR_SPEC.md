## Executive Summary

v0.2 transforms AgentSoul from **optional memory tool** → **mandatory default behavior** for any long-running AI agent.

**The Problem We're Solving:**
- Developers *know* persistence is good, but don't use it consistently
- LLM context grows unchecked → tokens wasted → costs explode → hallucination increases
- No feedback loop → no incentive to change behavior
- AgentSoul v0.1 works perfectly, but it's not being *used* as default

**The Solution:**
`agentsoul.orchestrator.PersistentAgent` — a drop-in agent wrapper that enforces good persistence patterns automatically and shows visible wins.

## Core Class: `PersistentAgent`

### Initialization
```python
from agentsoul.orchestrator import PersistentAgent

agent = PersistentAgent(
    agent_id="hvac_clerk_001",              # unique identifier
    backend="pocketbase",                   # "pocketbase" | "sqlite" | "rest"
    backend_url="http://127.0.0.1:8090",   # if not local
    max_context_tokens=8000,                # hard limit (fail if exceeded)
    auto_persist_every=5,                   # persist every N turns OR M minutes
    feedback=True,                          # show token/cost savings
    encryption_passphrase="secure_123"      # for export
)
```

### Core Loop (The Pattern)
```python
# This loop enforces good behavior by default
for task in incoming_tasks:
    
    # 1. Load state (always first)
    state = agent.load_state()
    
    # 2. Run your actual logic (armed with prior context)
    result = agent.run(task, context=state)
    
    # 3. Persist (always last)
    agent.persist(result)
    
    # 4. Show feedback (what did we save?)
    agent.print_feedback()
```

### Public Methods

#### `load_state() -> Dict[str, Any]`
- Queries PocketBase for prior agent state
- Returns: `{"task_history": [...], "decision_log": [...], "context_trim_point": 0.6}`
- If no prior state exists, returns empty dict
- **Never fails** — graceful degradation

#### `run(task: str, context: Dict = None) -> Dict`
- Executes the agent's core logic (your code goes here as a callback)
- Tracks tokens used (via LLM call hook)
- Auto-trims context if approaching `max_context_tokens`
- Returns: `{"output": "...", "tokens_used": 1247, "context_before": 7800, "context_after": 2100}`

#### `persist(result: Dict) -> bool`
- Writes result to PocketBase
- Logs to audit trail
- Returns success/failure
- **Never blocks** — fire-and-forget async-ready

#### `print_feedback() -> None`
- Prints pretty box showing:
  ```
  ┌─ Turn #23 Complete
  │  Tokens used: 1,247
  │  Context before: 7,800 (97%)
  │  Context after: 2,100 (26%)
  │  Tokens saved this turn: 5,700
  │  Cost saved: $0.0057 (at $0.001/1K tokens)
  │  Persistence: ✅ Written to PocketBase
  └─ Total saved this session: $0.0342
  ```

#### `export_soul() -> str`
- Exports encrypted artifact (full agent identity + all state)
- Returns file path to encrypted JSON
- Ready for transfer to new hardware

#### `trim_context(target_percent: float = 0.4) -> int`
- Manually trim context to target % of max
- Returns: tokens freed
- Used internally when approaching limits
- Also exposed for manual calls

## Behavior: Automatic Context Management

### Auto-Trim Logic
```python
# Inside agent.run() after each turn:

if current_context_percent > 0.85:
    # ⚠ We're getting full. Time to trim.
    agent.trim_context(target_percent=0.4)
    
    # Log this to audit trail
    agent.soul.log_interaction(
        "auto_trim",
        "context_management",
        {
            "triggered_at": 0.85,
            "trimmed_to": 0.4,
            "tokens_freed": 3200
        }
    )
    
    print("⚠  Context trim triggered (was 85%, now 40%)")
```

### Persistence Frequency
```python
# Persist every N turns
if turn_count % auto_persist_every == 0:
    agent.persist()

# OR persist on time interval
if time.time() - last_persist_time > (auto_persist_every * 60):
    agent.persist()
```

## Integration: How Developers Use It

### Before (v0.1 — Optional)
```python
from agentsoul import AgentSoul

# They *could* do this, but often don't
soul = AgentSoul.from_pocketbase(...)
memory = soul.recall(...)
# ... work ...
soul.remember(...)  # sometimes forgotten
```

### After (v0.2 — Default)
```python
from agentsoul.orchestrator import PersistentAgent

agent = PersistentAgent(agent_id="hvac_001")

for task in task_queue:
    state = agent.load_state()
    result = agent.run(task, context=state)
    agent.persist(result)
    agent.print_feedback()  # instant visibility
```

**The difference:** In v0.1, persistence is optional. In v0.2, it's structural.

## Feedback System (Why This Matters)

### Token Savings Calculation
```python
# Before trim
context_used = 7800 tokens

# After trim
context_used = 2100 tokens

# Savings
saved = 7800 - 2100 = 5700 tokens

# Cost impact (at $0.001 per 1K tokens)
cost_saved = (5700 / 1000) * $0.001 = $0.0057

# Print this every turn
print(f"💾 Tokens saved: {saved:,} (${cost_saved:.4f} at current rates)")
```

**Why this works:**
- Agents (and humans) respond to visible wins
- Showing "$0.0057 saved this turn" makes persistence feel *real*
- Over a 100-task session: $0.57 saved becomes obvious
- Multiplied across 10 agents in a company: suddenly it's $5.70/session = material
